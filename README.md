# simple HTTP server
<body>

## 📋 Overview

This is a minimal Express.js server that:
- Listens on port 3001
- Responds with "Hello express JS" to root (`/`) GET requests
- Logs server status to console


## 🛠️ Installation & Usage

Prerequisites:
<ol>
  <li>Node.js (v18 or later)</li>
  <li>npm/yarn</li>
</ol>
